# 🧮 React Calculator Project

A clean and responsive **Calculator Application** built by **SAROJ DHITAL** using **React.js** and deployed on **Vercel**.

🔗 **Live Demo:**  
https://react-calculator-project-saroj.vercel.app/

This application performs standard arithmetic operations with a modern UI, dark mode support, and calculation history.

---

## 🛠 Languages & Technologies Used

### 💻 Languages
- JavaScript (ES6+)
- HTML5
- CSS3

### ⚙️ Technologies & Tools
- React.js (Component-based architecture)
- React Hooks (useState, useEffect)
- Node.js (Development environment)
- npm (Package manager)
- Vercel (Deployment platform)

---

## 🚀 Core Features

### ✅ Basic Arithmetic Operations
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

### ✅ Advanced Functionalities
- Square root (√)
- Percentage (%)
- Power (^)
- Plus/Minus toggle (+/−)

### ✅ Memory Operations
- M+
- M−
- MR (Memory Recall)
- AC / Clear

### ✅ Dark Mode
- Toggle between Light and Dark theme
- Clear and readable display in both modes

### ✅ Calculation History
- Stores recent calculations
- Displays previous results for reference

### ✅ Responsive Design
- Works smoothly on desktop and mobile
- Clean and centered layout

---

## 📘 How It Works

1. Enter numbers using on-screen buttons or keyboard.
2. Select an operation.
3. Press `=` to calculate the result.
4. View calculation history below the calculator.
5. Toggle dark mode if preferred.

## 💻 Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/Joras-Latihd/React_Calculator_Project.git
   ```

2. Navigate to the project folder:

   ```bash
   cd React_Calculator_Project
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open in browser:

   ```
   http://localhost:3000
   ```

---

## 🌐 Deployment

This project is deployed using **Vercel**.

To deploy your own version:

1. Push the project to GitHub.
2. Connect the repository to Vercel.
3. Deploy from the Vercel dashboard.
